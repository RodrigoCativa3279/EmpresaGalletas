USO DE LA PÁGINA

Estado del producto: Se debe registrar el estado del producto en el sector correspondiente para estar al tanto de problemas y deficiencias.

Estado del camión: Se debe avisar cuando el camión ha llegado al punto de entrega y establecer un código UNICO para cada punto de venta antes de partir, para evitar robos o entregas incorrectas. Se debe enviar el código UNICO al cliente correspondiente.
